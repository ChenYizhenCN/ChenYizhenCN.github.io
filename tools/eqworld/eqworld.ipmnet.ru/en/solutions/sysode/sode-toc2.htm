2. Linear Systems of Three or More Ordinary Differential Equations; x = x(t), y = y(t), z = z(t)

  1. x′ = ax,   y′ = bx + cy,   z′ = dx + ky + pz.
  2. x′ = cy − bz,   y′ = az − cx,   z′ = bx − ay.
  3. ax′ = bc(y − z),   by′ = ac(z − x),   cz′ = ab(x − y).
  4. x′ = (a1f + g)x + a2fy + a3fz,   y′ = b1fx + (b2f + g)y + b3fz,   z′ = c1fx + c2fy + (c3f + g)z.
  5. x′ = h(t)y − g(t)z,   y′ = f(t)z − h(t)x,   z′ = g(t)x − f(t)y.
  6. x′k = ak1x1 + ak2x2 + … + aknxn;   k = 1, 2, …, n.
