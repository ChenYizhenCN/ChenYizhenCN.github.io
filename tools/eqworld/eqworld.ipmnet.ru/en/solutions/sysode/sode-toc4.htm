4. Nonlinear Systems of Three or More Ordinary Differential Equations

4.1. Systems of First-Order Ordinary Differential Equations; x = x(t), y = y(t), z = z(t)

  1. ax′ = (b − c)yz,   by′ = (c − a)zx,   cz′ = (a − b)xy.
  2. ax′ = (b − c)yzf(x, y, z, t),   by′ = (c − a)zxf(x, y, z, t),   cz′ = (a − b)xyf(x, y, z, t).
  3. x′ = a(y − x),   y′ = bx − y − xz,   z′ = −cz + xy.    Lorenz equations.
  4. x′ = cF2 − bF3,   y′ = aF3 − cF1,   z′ = bF1 − aF2,   where   Fn = Fn(x, y, z, t).
  5. x′ = czF2 − byF3,   y′ = axF3 − czF1,   z′ = byF1 − axF2,   where   Fn = Fn(x, y, z, t).
  6. x′ = x(cF2 − bF3),   y′ = y(aF3 − cF1),   z′ = z(bF1 − aF2),   where   Fn = Fn(x, y, z, t).
  7. x′ = h(z)F2 − g(y)F3,   y′ = f(x)F3 − h(z)F1,   z′ = g(y)F1 − f(x)F2   where   Fn = Fn(x, y, z, t).

4.2. Systems of Second-Order Ordinary Differential Equations; x = x(t), y = y(t), z = z(t)

  1. x″ = Fx,   y″ = Fy,   z″ = Fz,   where   F = F(r),   r = (x2 + y2 + z2)1/2.
  2. x″ = xF,   y″ = yF,   z″ = zF,   where   F = F(x, y, z, t, x′, y′, z′).
  3. x″ = F1,   y″ = F2,   z″ = F3,   where   Fn = Fn(t, tx′ − x, ty′ − y, tz′ − z).
  4. x″ = cF2 − bF3,   y″ = aF3 − cF1,   z″ = bF1 − aF2,   where   F = F(x, y, z, t, x′, y′, z′).
