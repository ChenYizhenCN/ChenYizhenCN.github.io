5. Third-Order Partial Differential Equations

  1. wt + wxxx − 6wwx = 0.    Korteweg--de Vries equation.
  2. wt + wxxx − 6wwx + (2t)−1w = 0.    Cylindrical Korteweg--de Vries equation.
  3. wt + wxxx + 6σw2wx = 0.    Modified Korteweg--de Vries equation.
  4. wt + wxxx + f(w)wx = 0.    Generalized Korteweg--de Vries equation.
  5. wywxy − wxwyy = awyyy.    Hydrodynamic boundary layer equation.
  6. wywxy − wxwyy = awyyy + f(x).    Boundary layer equation with pressure gradient.
