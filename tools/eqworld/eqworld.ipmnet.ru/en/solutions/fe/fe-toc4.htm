4. Nonlinear Functional Equations with Several Independent Variables

  1. f(x + y) = f(x)f(y).    Cauchy's exponential equation.
  2. f(xy) = f(x)f(y).    Cauchy's power equation.
  3. f(y + x)+f(y − x) = 2f(x)f(y).    D'Alembert equation.
  4. f(x + y) = axyf(x)f(y).
  5. f(x + y) = f(x) + f(y) − af(x)f(y).    Equation of the theory probability.
  6. f(x)g(y) = h(x + y).
  7. f(x)g(y) + h(y) = f(x + y).
  8. f(x + y)f(x − y) = f 2(x).    Lobachevsky equation.
  9. (f 2(x)/2 + f 2(y)/2)1/2 = f((x2/2 + y2/2)1/2).
  10. f(x, y)f(y, z) = f(x, z).    Cantor's second equation.
  11. M(f(x), f(y)) = f(M(x, y)).
  12. f1(x)g1(y) + f2(x)g2(y) + f3(x)g3(y) = 0.    Bilinear functional equation.
  13. f1(x)g1(y) + f2(x)g2(y) + f3(x)g3(y) + f4(x)g4(y) = 0.    Bilinear functional equation.
  14. f(x) + g(y) = Q(z),   where    z = φ(x) + ψ(y).
  15. f(t) + g(x) + h(x)Q(z) + R(z) = 0,   where    z = φ(x) + ψ(t).
  16. f(t) + g(x)Q(z) + h(x)R(z) = 0,   where    z = φ(x) + ψ(t).
