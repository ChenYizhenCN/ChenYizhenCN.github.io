Las ecuaciones desempeñan un papel crucial en las matemáticas modernas y forman la base para la modelación matemática de numerosos fenómenos y procesos en las ciencias y las ingenierías.

El sitio web científico-edicativo internacional EqWorld presenta información extensa sobre soluciones de varias clases de ecuaciones matemáticas: ecuaciones diferenciales ordinarias, ecuaciones diferenciales parciales, ecuaciones integrales, ecuaciones funcionales, y otras ecuaciones matemáticas. También describe algunos métodos para resolver ecuaciones, incluye artículos cietíficos interesantes, presenta enlaces a sitios matemáticos y software matemático, listas de manuales, libros de texto, y monografías de gran utilidad, y hace referencia a editoriales científicas, revistas, entre otros. El sitio web incluye una sección dinámica EqArchive que permite a los autores publicar rápidamente sus ecuaciones (diferenciales, integrales, y otros) y también soluciones exactas, primeras integrales, y transformadas.

El sitio web EqWorld está pensado para investigadores, profesores universitarios, ingenieros, y estudiantes en todo el mundo. EqWorld contiene cerca de 2000 páginas web y se visita más de 3000 usuarios al día (de más de 200 países del mundo). Todos los recursos que se presentan en este sitio son gratuitos para sus usuarios.

El sitio web EqWorld presenta información extensa acerca de ecuaciones diferenciales ordinarias, ecuaciones diferenciales parciales, ecuaciones integrales, ecuaciones funcionales, y otras ecuaciones matemáticas.
