2. Линейные уравнения второго порядка гиперболического типа

  1. wtt = a2wxx.    Волновое уравнение (уравнение колебаний струны).
  2. wtt = a2wxx + Φ(x, t).    Неоднородное волновое уравнение.
  3. wtt = a2wxx − bw.    Уравнение Клейна--Гордона.
  4. wtt = a2wxx − bw + Φ(x, t).    Неоднородное уравнение Клейна--Гордона.
  5. wtt = a2(wrr + r−1wr) + Φ(r, t).    Неоднородное волновое уравнение с осевой симметрией.
  6. wtt = a2(wrr+2r−1wr) + Φ(r, t).    Неоднородное волновое уравнение с центральной симметрией.
  7. wtt + kwt = a2wxx + bw.    Телеграфное уравнение.
