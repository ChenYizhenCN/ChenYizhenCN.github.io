1. Линейные уравнения второго порядка параболического типа

  1. wt = awxx.    Уравнение теплопроводности (линейное уравнение теплопроводности).
  2. wt = awxx + Φ(x, t).    Неоднородное уравнение теплопроводности (уравнение теплопроводности с источником).
  3. wt = awxx + bwx + cw + Φ(x, t).    Уравнение конвективной теплопроводности с источником.
  4. wt = a(wrr + r−1wr).    Уравнение теплопроводности с осевой симметрией.
  5. wt = a(wrr + r−1wr) + Φ(r, t).    Неоднородное уравнение теплопроводности с осевой симметрией.
  6. wt = a(wrr + 2r−1wr).    Уравнение теплопроводности с центральной симметрией.
  7. wt = a(wrr + 2r−1wr) + Φ(r, t).    Неоднородное уравнение теплопроводности с центральной симметрией.
  8. iwt = −kwxx + U(x)w.    Уравнение Шредингера (линейное уравнение Шредингера).
