1. Линейные разностные и функциональные уравнения с одной независимой переменной

1.1. Линейные разностные и функциональные уравнения, содержащие неизвестную функцию с двумя различными аргументами

Линейные разностные уравнения первого порядка

  1. y(x + 1) − ay(x) = 0.    Линейное однородное разностное уравнение первого порядка с постоянными коэффициентами.
  2. y(x + 1) − ay(x) = f(x).    Линейное неоднородное разностное уравнение первого порядка с постоянными коэффициентами.
  3. y(x + 1) − xy(x) = 0.
  4. y(x + 1) − a(x − b)(x − c)y(x) = 0.
  5. y(x + 1) − R(x)y(x) = 0,   где  R(x) -- рациональная функция.
  6. y(x + 1) − f(x)y(x) = 0.
  7. y(x + a) − by(x) = 0.
  8. y(x + a) − by(x) = f(x).
  9. y(x + a) − bxy(x) = 0.
  10. y(x + a) − f(x)y(x) = 0.

Линейные функциональные уравнения, содержащие y(x) и y(ax)

  1. y(ax) − by(x) = 0.
  2. y(ax) − by(x) = f(x).

Линейные функциональные уравнения, содержащие y(x) и y(a − x)

  1. y(x) − y(a − x) = 0.
  2. y(x) + y(a − x) = 0.
  3. y(x) + y(a − x) = b.
  4. y(x) + y(a − x) = f(x).
  5. y(x) − y(a − x) = f(x).
  6. y(x) + g(x)y(a − x) = f(x).

Линейные функциональные уравнения, содержащие y(x) и y(z), где z = φ(x)

  1. y(xa) − by(x) = 0.
  2. y(x) − y(a/x) = 0.
  3. y(x) + y(a/x) = 0.
  4. y(x) + y(a/x) = b.
  5. y(x) + y(a/x) = f(x).
  6. y(x) − y(a/x) = f(x).
  7. y(x) + g(x)y(a/x) = f(x).
  8. y(x) − y((a − x)/(1 + bx)) = 0.
  9. y(x) + y((a − x)/(1 + bx)) = 0.
  10. y(x) + y((a − x)/(1 + bx)) = f(x).
  11. y(x) − y((a − x)/(1 + bx)) = f(x).
  12. y(x) − cy((a − x)/(1 + bx)) = f(x).
  13. y(x) + g(x)y((a − x)/(1 + bx)) = f(x).
  14. y(x) + cy((ax − β)/(x + b)) = f(x),    β = a2 + ab + b2.
  15. y(x) + cy((bx + β)/(a − x)) = f(x),    β = a2 + ab + b2.
  16. y(x) + g(x)y((ax − β)/(x + b)) = f(x),    β = a2 + ab + b2.
  17. y(x) + g(x)y((bx + β)/(a − x)) = f(x),    β = a2 + ab + b2.
  18. y(x) − y((a2 − x2)1/2) = 0.
  19. y(x) + y((a2 − x2)1/2) = 0.
  20. y(x) + y((a2 − x2)1/2) = b.
  21. y(x) + y((a2 − x2)1/2) = f(x).
  22. y(x) − y((a2 − x2)1/2) = f(x).
  23. y(x) + g(x)y((a2 − x2)1/2) = f(x).

Линейные функциональные уравнения, содержащие y(sin x) и y(cos x)

  1. y(sin x) − y(cos x) = 0.
  2. y(sin x) + y(cos x) = 0.
  3. y(sin x) + y(cos x) = a.
  4. y(sin x) + y(cos x) = f(x).
  5. y(sin x) − y(cos x) = f(x).
  6. y(sin x) + g(x)y(cos x) = f(x).

Линейные функциональные уравнения, содержащие y(x) и y(ω(x)), где ω(ω(x)) = x

  1. y(x) − y(ω(x)) = 0,    где   ω(ω(x)) = x.
  2. y(x) + y(ω(x)) = 0,    где   ω(ω(x)) = x.
  3. y(x) + y(ω(x)) = b,    где   ω(ω(x)) = x.
  4. y(x) + y(ω(x)) = f(x),    где   ω(ω(x)) = x.
  5. y(x) − y(ω(x)) = f(x),    где   ω(ω(x)) = x.
  6. y(x) + g(x)y(ω(x)) = f(x),    где   ω(ω(x)) = x.

1.2. Другие линейные разностные и функциональные уравнения

Линейные разностные уравнения второго порядка, yn = y(n)

  1. yn+2 + ayn+1 + byn = 0.    Линейное однородное разностное уравнение второго порядка с постоянными коэффициентами.
  2. yn+2 + ayn+1 + byn = fn.    Линейное неоднородное разностное уравнение второго порядка с постоянными коэффициентами.
  3. y(x + 2) + ay(x + 1) + by(x) = 0.    Линейное однородное разностное уравнение второго порядка с постоянными коэффициентами.
  4. y(x + 2) + ay(x + 1) + by(x) = f(x).    Линейное неоднородное разностное уравнение второго порядка с постоянными коэффициентами.
  5. y(x + 2) + a(x + 1)y(x + 1) + bx(x + 1)y(x) = 0.

Другие функциональные уравнения

  1. Ay(ax) + By(bx) + y(x) = 0.
  2. Ay(xa) + By(xb) + y(x) = 0.
  3. y(y(x)) − x = 0.     Уравнение Беббиджа (Babbage).
  4. y(y(x)) + ay(x) + bx = 0.
  5. y(y(y(x))) − x = 0.
  6. Ay(x) + By((ax − β)/(x + b)) + Cy((bx + β)/(a − x)) = f(x),    β = a2 + ab + b2.
  7. f1(x)y(x) + f2(x)y((ax − β)/(x + b)) + f3(x)y((bx + β)/(a − x)) = g(x),    β = a2 + ab + b2.
  8. yn+m + am−1yn+m−1 + ... + a1yn+1 + a0yn = 0.    Линейное однородное разностное уравнение m-го порядка с постоянными коэффициентами.
  9. yn+m + am−1yn+m−1 + ... + a1yn+1 + a0yn = fn.    Линейное неоднородное разностное уравнение m-го порядка с постоянными коэффициентами.
  10. y(x + n) + an−1y(x + n − 1) + ... + a1y(x + 1) + a0y(x) = 0.    Линейное однородное разностное уравнение n-го порядка с постоянными коэффициентами.
  11. y(x + n) + an−1y(x + n − 1) + ... + a1y(x + 1) + a0y(x) = f(x).    Линейное неоднородное разностное уравнение m-го порядка с постоянными коэффициентами.
  12. y(x + bn) + an−1y(x + bn−1) + ... + a1y(x + b1) + a0y(x) = 0.
  13. ay(xα) + by(xβ) + cy(xσ) + ... + y(x) = 0.
  14. y(anx) + bn−1y(an−1x) + ... + b1y(a1x) + b0y(x) = 0.
  15. y[n](x) + an−1y[n−1](x) + ... + a1y(x) + a0x = 0,    y[n](x) = y(y[n−1](x)).
