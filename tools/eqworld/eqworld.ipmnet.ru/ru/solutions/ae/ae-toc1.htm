Алгебраические уравнения

  1. ax + b = 0.   Линейное уравнение.
  2. ax2 + bx + c = 0.   Квадратное уравнение.
  3. ax3 + bx2 + cx + d = 0.   Кубическое уравнение.
  4. ax4 + bx2 + c = 0.   Биквадратное уравнение.
  5. ax4 + bx3 + cx2 + bx + a = 0.   Возвратное (алгебраическое) уравнение.
  6. ax4 + bx3 + cx2 − bx + a = 0.   Модифицированное возвратное уравнение.
  7. ab2x4 + bx3 + cx2 + dx + ad2 = 0.   Обобщенное возвратное уравнение.
  8. ax4 + bx3 + cx2 + dx + e = 0.   Уравнение четвертой степени общего вида.
  9. xn − a = 0.   Двучленное алгебраическое уравнение n-й степени.
  10. ax2n + bxn + c = 0.
  11. a0x2n + a1x2n−1 + a2x2n−2 + ... + a2x2 + a1x + a0 = 0.   Возвратное (алгебраическое) уравнение.
  12. anxn + an−1xn−1 + ... + a1x + a0 = 0.   Алгебраическое уравнение n-й степени общего вида.

Веб-сайт EqWorld содержит обширную информацию о решениях различных классов обыкновенных дифференциальных уравнений, дифференциальных уравнений с частными производными (уравнений математической физики), интегральных уравнений, функциональных уравнений и других математических уравнений.
