3. Нелинейные системы двух обыкновенных дифференциальных уравнений

3.1. Cистемы дифференциальных уравнений первого порядка; x = x(t), y = y(t)

  1. x′ = xnF(x, y),   y′ = g(y) F(x, y).
  2. x′ = eλxF(x, y),   y′ = g(y) F(x, y).
  3. x′ = F(x, y),   y′ = G(x, y).    Автономная система общего вида.
  4. x′ = f1(x) g1(y) Φ(x, y, t),   y′ = f2(x) g2(y) Φ(x, y, t).
  5. x = tx′ + F(x′, y′),   y = ty′ + G(x′, y′).    Система уравнений Клеро.

3.2. Cистемы дифференциальных уравнений второго порядка; x = x(t), y = y(t)

  1. x″ = xf(ax − by) + g(ax − by),   y″ = yf(ax − by) + h(ax − by).
  2. x″ = xf(y/x),   y″ = yg(y/x).
  3. x″ = kxr−3,   y″ = kyr−3;    r = (x2 + y2)1/2.    Уравнения движения материальной точки в гравитационном поле.
  4. x″ = xf(r),   y″ = yf(r);    r = (x2 + y2)1/2.    Уравнения движения материальной точки в поле центральных сил.
  5. x″ = xf(x2 + y2, y/x) − yg(y/x),   y″ = yf(x2 + y2, y/x) + xg(y/x).
  6. x″ = −f(y)g(v)x′,   y″ = −f(y)g(v)y′ − a;    v = [(x′)2 + (y′)2]1/2.    Уравнения движения снаряда в воздухе.
  7. x″ + a(t)x = x−3f(y/x),   y″ + a(t)y = y−3g(y/x).    Обобщенная система Ермакова.
  8. x″ = x−3F(x/φ(t), y/φ(t)),   y″ = y−3G(x/φ(t), y/φ(t));    φ(t) = (at2 + bt + c)1/2.
  9. x″ = f(y′/x′),   y″ = g(y′/x′).
  10. x″ = xΦ(x, y, t, x′, y′),   y″ = yΦ(x, y, t, x′, y′).
  11. x″ + x−3f(y/x) = xΦ(x, y, t, x′, y′),   y″ + y−3g(y/x) = yΦ(x, y, t, x′, y′).
  12. x″ = F(t, tx′ − x, ty′ − y),   y″ = G(t, tx′ − x, ty′ − y).
  13. x″ = x′Φ(x, y, t, x′, y′) + f(y),   y″ = −y′Φ(x, y, t, x′, y′) + g(x).
  14. x″ = ay′Φ(x, y, t, x′, y′) + f(x),   y″ = bx′Φ(x, y, t, x′, y′) + g(y).
  15. x″ = f(y′)Φ(x, y, t, x′, y′),   y″ = g(x′)Φ(x, y, t, x′, y′).
