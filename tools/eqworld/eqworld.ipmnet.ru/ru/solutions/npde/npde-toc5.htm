5. Нелинейные уравнения третьего порядка

  1. wt + wxxx − 6wwx = 0.    Уравнение Кортевега--де Фриза.
  2. wt + wxxx − 6wwx + (2t)−1w = 0.    Цилиндрическое уравнение Кортевега--де Фриза.
  3. wt + wxxx + 6σw2wx = 0.    Модифицированное уравнение Кортевега--де Фриза.
  4. wt + wxxx + f(w)wx = 0.    Обобщенное уравнение Кортевега--де Фриза.
  5. wywxy − wxwyy = awyyy.    Уравнение гидродинамического пограничного слоя.
  6. wywxy − wxwyy = awyyy + f(x).    Уравнение гидродинамического пограничного слоя с градиентом давления.
