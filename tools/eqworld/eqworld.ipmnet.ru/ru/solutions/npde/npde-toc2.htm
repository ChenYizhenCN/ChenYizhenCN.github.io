2. Нелинейные уравнения второго порядка гиперболического типа

2.1. Нелинейные волновые уравнения вида wtt = awxx + f(w)

  1. wtt = awxx + aw + bwn.    Уравнение Клейна--Гордона со степенной нелинейностью.
  2. wtt = wxx + awn + bw2n−1.    Уравнение Клейна--Гордона со степенной нелинейностью.
  3. wtt = a2wxx + beβw.    Модифицированное уравнение Лиувилля.
  4. wtt = wxx + aeβw + be2βw.    Уравнение Клейна--Гордона с экспоненциальной нелинейностью.
  5. wtt = awxx + b sh(λw).    Уравнение sh-Гордона.
  6. wtt = awxx + b sin(λw).    Уравнение синус-Гордона.
  7. wtt = wxx + f(w).    Нелинейное уравнение Клейна--Гордона общего вида.

2.2. Другие нелинейные уравнения гиперболического типа

  1. wtt = a(wwx)x.
  2. wtt = a(wnwx)x + bwk.
  3. wtt = a(eλwwx)x.
  4. wtt = ax−n(xnwx)x + f(w).
  5. wtt = [a(x + b)nwx]x + f(w).
  6. wtt = a(eλxwx)x + f(w).
  7. wtt = [f(w)wx]x.
