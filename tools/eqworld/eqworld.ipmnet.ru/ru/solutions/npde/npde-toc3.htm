3. Нелинейные уравнения второго порядка эллиптического типа

3.1. Стационарные уравнения нелинейной теплопроводности вида wxx + wyy = f(w)

  1. wxx + wyy = aw + bwn.
  2. wxx + wyy = awn + bw2n−1.
  3. wxx + wyy = aeβw.
  4. wxx + wyy = aeβw + be2βw.
  5. wxx + wyy = aw ln(βw).
  6. wxx + wyy = a sin(βw).
  7. wxx + wyy = f(w).

3.2. Уравнения анизотропной теплопроводности вида [f(x)wx]x + [g(y)wy]y = h(w)

  1. (axnwx)x + (bymwy)y = f(w).
  2. awxx + (beμywy)y = f(w).
  3. (aeβxwx)x + (beμywy)y = f(w).
  4. [f(x)wx]x + [g(y)wy]y = kw ln w.

3.3. Уравнения анизотропной теплопроводности вида [f(w)wx]x + [g(w)wy]y = h(w)

  1. wxx + [(αw + β)wy]y = 0.    Стационарное уравнение Хохлова--Заболоцкой.
  2. wxx + (aeβwwy)y = 0.
  3. [f(w)wx]x + [g(w)wy]y = 0.
