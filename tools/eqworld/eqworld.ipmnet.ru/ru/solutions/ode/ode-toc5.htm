5. Нелинейные обыкновенные дифференциальные уравнения старших порядков

  1. y′′′ = Axαyβ.    Уравнение Эмдена--Фаулера третьего порядка.
  2. y′′′ = ay−5/2 + by−7/2.
  3. y′′′ = f(y).
  4. yy′′′ = f(x).
  5. y′′′′ = Ay−5/3.
  6. y′′′′ = f(y).
  7. F(x, y′, y′′, ..., y(n)) = 0.    Уравнение не зависит явно от y.
  8. F(y, y′, y′′, ..., y(n)) = 0.    Автономное уравнение.
  9. F(x, xy′′ − my, y(m+1), y(m+2), ..., y(n)) = 0,    m = 1, 2, ..., n − 1.
  10. F(xkym, xy′/y, x2y′′/y, ..., xny(n)/y) = 0.    Обобщенно-однородное уравнение.
  11. F(eαxym, y′/y, y′′/y, ..., y(n)/y) = 0.
  12. F(xmeαy, xy′, x2y′′, ..., xny(n)) = 0.
