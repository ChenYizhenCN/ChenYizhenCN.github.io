1. Обыкновенные дифференциальные уравнения первого порядка

  1. y′ = f(y).    Автономное уравнение.
  2. y′ = f(x)g(y).    Уравнение с разделяющимися переменными.
  3. g(x)y′ = f1(x)y + f0(x).    Линейное уравнение.
  4. g(x)y′ = f1(x)y + fn(x)yn.    Уравнение Бернулли.
  5. y′ = f(y/x).    Однородное уравнение.
  6. y′ = ay2 + bxn.    Специальное уравнение Риккати.
  7. y′ = y2 + f(x)y − a2 − af(x).    Уравнение Риккати - 1.
  8. y′ = f(x)y2 + ay − ab − b2f(x).    Уравнение Риккати - 2.
  9. y′ = y2 + xf(x)y + f(x).    Уравнение Риккати - 3.
  10. y′ = f(x)y2 − axnf(x)y + anxn−1.    Уравнение Риккати - 4.
  11. y′ = f(x)y2 + anxn−1 − a2x2nf(x).    Уравнение Риккати - 5.
  12. y′ = −(n + 1)xny2 + xn+1f(x)y − f(x).    Уравнение Риккати - 6.
  13. xy′ = f(x)y2 + ny + ax2nf(x).    Уравнение Риккати - 7.
  14. xy′ = x2nf(x)y2 + [axnf(x) − n]y + bf(x).    Уравнение Риккати - 8.
  15. y′ = f(x)y2 + g(x)y − a2f(x) − ag(x).    Уравнение Риккати - 9.
  16. y′ = f(x)y2 + g(x)y + anxn−1 − a2x2nf(x) − axng(x).    Уравнение Риккати - 10.
  17. y′ = aeλxy2 + aeλxf(x)y + λf(x).    Уравнение Риккати - 11.
  18. y′ = f(x)y2 − aeλxf(x)y + aλeλx.    Уравнение Риккати - 12.
  19. y′ = f(x)y2 + aλeλx − a2e2λxf(x).    Уравнение Риккати - 13.
  20. y′ = f(x)y2 + λy + ae2λxf(x).    Уравнение Риккати - 14.
  21. y′ = y2 − f2(x) + f′(x).    Уравнение Риккати - 15.
  22. y′ = f(x)y2 − f(x)g(x)y + g′(x).    Уравнение Риккати - 16.
  23. y′ = f(x)y2 + g(x)y + h(x).    Общеее уравнение Риккати.
  24. yy′ = y + f(x).    Уравнение Абеля второго рода в канонической форме.
  25. yy′ = f(x)y + g(x).    Уравнение Абеля второго рода.
  26. yy′ = f(x)y2 + g(x)y + h(x).    Уравнение Абеля второго рода.
  27. y′ = f(ax + by + c).
  28. y′ = f(y + axn + b) − anxn−1.
  29. y′ = (y/x)f(xnym).    Обобщенно-однородное уравнение.
  30. y′ = −(n/m)(y/x) + ykf(x)g(xnym).
  31. y′ = f((ax + by + c)/(αx + βy + γ)).
  32. y′ = xn−1y1−mf(axn + bym).
  33. [xnf(y) + xg(y)]y′ = h(y).
  34. x[f(xnym) + mxkg(xnym)]y′ = y[h(xnym) − nxkg(xnym)].
  35. x[f(xnym) + mykg(xnym)]y′ = y[h(xnym) − nykg(xnym)].
  36. x[sf(xnym) − mg(xkys)]y′ = y[ng(xkys) − kf(xnym)].
  37. [f(y) + amxnym−1]y′ + g(x) + anxn−1ym = 0.
  38. y′ = e−λxf(eλxy).
  39. y′ = eλyf(eλyx).
  40. y′ = yf(eαxym).
  41. y′ = x−1f(xneαy).
  42. y′ = f(x)eλy + g(x).
  43. y′ = −nx−1 + f(x)g(xney).
  44. y′ = −(α/m)y + ykf(x)g(eαxym).
  45. y′ = eαx−βyf(aeαx + beβy).
  46. [eαxf(y) + aβ]y′ + eβyg(x) + aα = 0.
  47. x[f(xneαy) + αyg(xneαy)]y′ = h(xneαy) − nyg(xneαy).
  48. [f(eαxym) + mxg(eαxym)]y′ = y[h(eαxym) − αxg(eαxym)].
