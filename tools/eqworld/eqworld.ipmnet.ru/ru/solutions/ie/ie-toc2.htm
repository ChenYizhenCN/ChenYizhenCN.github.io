2. Интегральные уравнения Вольтерра второго рода

2-1. Интегральные уравнения, ядра которых содержат степенные функции

  1. y(x) − λ y(t) dt = f(x).
  2. y(x) + λ (x − t)y(t) dt = f(x).
  3. y(x) + λ (x − t)2y(t) dt = f(x).
  4. y(x) + λ (x − t)3y(t) dt = f(x).
  5. y(x) + A (x − t)ny(t) dt = f(x).
  6. y(x) + λ (x − t)−1/2y(t) dt = f(x).    Уравнение Абеля второго рода.
  7. y(x) − λ (x − t)−αy(t) dt = f(x).    Обобщенное уравнение Абеля второго рода.

2-2. Интегральные уравнения, ядра которых содержат экспоненциальные функции

  1. y(x) + A eλ(x − t)y(t) dt = f(x).
  2. y(x) + A [eλ(x − t) − 1]y(t) dt = f(x).
  3. y(x) + A (x − t)eλ(x − t)y(t) dt = f(x).

2-3. Интегральные уравнения, ядра которых содержат гиперболические или специальные функции

  1. y(x) + A cosh[λ(x − t)]y(t) dt = f(x).
  2. y(x) + A sinh[λ(x − t)]y(t) dt = f(x).
  3. y(x) − λ J0(x − t)y(t) dt = f(x).

2-4. Интегральные уравнения, ядра которых содержат произвольные функции

  1. y(x) − g(x)h(t)y(t) dt = f(x).
  2. y(x) + (x − t)g(x)y(t) dt = f(x).
  3. y(x) + (x − t)g(t)y(t) dt = f(x).
  4. y(x) + K(x − t)y(t) dt = f(x).    Уравнение восстановления.
