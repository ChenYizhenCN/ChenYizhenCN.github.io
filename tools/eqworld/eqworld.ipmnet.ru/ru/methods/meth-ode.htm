Обыкновенные дифференциальные уравнения

Основные свойства и методы решения обыкновенных дифференциальных уравнений (В.Ф. Зайцев, А.Д. Полянин)

  • Линейные обыкновенные дифференциальные уравнения второго порядка
  • Нелинейные обыкновенные дифференциальные уравнения второго порядка

Симметрии обыкновенных дифференциальных уравнений

  • Решение дифференциальных уравнений с помощью инвариантов (А. Д. Полянин, К. Е. Казаков)
  • Опыт группового анализа обыкновенных дифференциальных уравнений (Н.Х. Ибрагимов, Знание, 1991)
  • Симметрия алгебраических и дифференциальных уравнений (Н.А. Кудряшов, Соросовский образовательный журнал, 1998, N 9)

Численные методы решения обыкновенных дифференциальных уравнений (А.И. Лобанов, И.Б. Петров)

  • Численные методы решения задачи Коши для систем обыкновенных дифференциальных уравнений
  • Численные методы решения жестких систем обыкновенных дифференциальных уравнений
  • Численное решение краевых задач для систем обыкновенных дифференциальных уравнений
 

Веб-сайт EqWorld содержит обширную информацию о решениях различных классов обыкновенных дифференциальных уравнений, дифференциальных уравнений с частными производными (уравнений математической физики), интегральных уравнений, функциональных уравнений и других математических уравнений.
