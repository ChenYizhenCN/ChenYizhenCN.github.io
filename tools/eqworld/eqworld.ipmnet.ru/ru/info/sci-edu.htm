Международные индексы цитирования научных журналов

Импакт фактор от Clarivate Analytics (бывший Thomson Reuters)

  • ISI Web of Knowledge — Journal Citation Reports: http://admin-apps.webofknowledge.com/JCR/JCR (доступ открыт только с компьютеров организаций, подключенных к базе ISI)
    • после первого входа по ссылке нажать "establish a new session";
    • список всех журналов или поиск конкретного журнала по англоязычному названию или ISSN англоязычной печатной версии.

  • Русскоязычный сайт Web of Science (раздел редакторам) http://wokinfo.com/russian/editors/ – представлен список журналов из России и стран СНГ в Web of Science Core Collection (формат Excel).

Другие международные индексы цитирования научных журналов

  • Scopus: http://www.scopus.com (доступ открыт только с компьютеров организаций, подключенных к базе Scopus).
  • Scopus Source List by Elsevier: https://www.elsevier.com/solutions/scopus/content
    • в разделе "What content is included in Scopus?" см ссылку "Scopus Source List" на Excel-файл с переченем журналов, индексирующихся в Scopus (полная информация, в том числе индексы SNIP, IPP, SJR, дата вхождений/выпадания из рейтинга и т.п.)
  • Elsevier в России – Scopus http://elsevierscience.ru/products/scopus/ – в формате Excel
    • список журналов, индексируемых в Scopus;
    • список российских журналов, индексируемых в Scopus;
    • список книг, индексируемых в Scopus.
  • JournalMetrics (IPP, SNIP & SJR) by Scopus (from Elsevier): http://www.journalmetrics.com/
  • SJR: Scientific Journal Rankings: http://www.scimagojr.com/journalsearch.php

Российские индексы цитирования научных журналов

  • Российский индекс научного цитирования (РИНЦ): http://elibrary.ru/titles.asp

    • после выполнения поиска рядом с количеством статей нажмите на пиктограмме для перехода к Анализу публикационной активности журнала
  • Импакт-факторы 2007: doc (306 Кбайт)

Индексы цитирования авторов

  • Российский индекс научного цитирования (РИНЦ): http://elibrary.ru/authors.asp
    • альтернативный вход: http://elibrary.ru ⇒ Российский индекс научного цитирования ⇒ Поиск авторов
    • после выполнения поиска справа нажмите на пиктограмме для перехода к Анализу публикационной активности автора
  • Google Scholar: http://scholar.google.com/schhp?hl=en&num=100
    • рекомендуется производить поиск с инициалами, например A.D. Polyanin
  • ISI Web of Knowledge - Cited Reference Search: http://apps.webofknowledge.com (доступ открыт только с компьютеров организаций, подключенных к базе ISI)
    • поиск выполнять по полю Author (затем возможно уточнение/сужение результатов – см. Refine слева);
    • после окончания поиска справа нажать ссылку Create Citation Report.
  • ResearcherID: http://researcherid.com - ресурс от Clarivate Analytics (создателя и владельца Web of Science) для создания персональных страниц исследователей и просмотра списка публикаций, индекса Хирша на основе публикаций входящих в Web of Science
    • найти (Search) и перейти в профиль интересующего автора;
    • перейти в раздел Citation Metrics (ссылка слева).
  • Scopus: http://www.scopus.com
    • полноценный доступ открыт только с компьютеров организаций, подключенных к базе Scopus;
    • с произвольных компьютеров доступен Предварительный поиск авторов (показывает последние 20 записей без возможности детального просмотра и редактирования: привязанные к автору публикации, число цитирований, h-index) – нажать на ссылку "Author Preview" (см. последний пункт на странице входа в Scopus или прямая ссылка: http://www.scopus.com/search/form/authorFreeLookup.url);
  • Индексы цитирования работ российских ученых: проект "Кто есть кто в российской науке" по лицензии Thomson Reuters http://expertcorps.ru/science/whoiswho
